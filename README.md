# ConverToDazz — Руководство пользователя

## Введение

**ConverToDazz** — удобная программа для конвертации **изображений** и **аудиофайлов**.  
Она позволяет быстро менять форматы файлов, сохранять их в нужной папке и получать информацию о свойствах файлов.

Программа поддерживает **локализацию**: интерфейс будет на русском, а если язык системы не русский — на английском.

---

## Основные возможности

### Конвертер изображений

- Поддерживаемые форматы: `jpg/jpeg`, `png`, `webp`, `avif`, `bmp`, `tiff`, `ico`  
- Автоматическое определение формата выбранного файла  
- Блокировка изменения формата после автоопределения  
- Превью изображения перед конвертацией  
- Отображение информации о файле:
  - Размер файла
  - Дата последнего изменения  
- Возможность изменить имя файла перед сохранением  
- Конвертация в ICO с набором стандартных размеров  
- Сохранение файла в выбранную папку  

### Конвертер аудио

- Поддерживаемые форматы: `mp3`, `wav`, `ogg`, `flac`, `aac`, `wma`  
- Автоопределение формата входного файла  
- Отображение информации о файле:
  - Имя файла
  - Длительность
  - Битрейт  
- Прогресс-бар показывает реальный прогресс конвертации  
- Сохранение результата в выбранное место  

---

## Начало работы

1. **Запуск программы**  
   Дважды кликните на `ConverToDazz.exe`. Откроется главное окно с двумя кнопками:  
   - **Конвертер изображений**  
   - **Конвертер аудио**

2. **Выбор модуля**  
   Нажмите на нужную кнопку, чтобы открыть соответствующий конвертер.  
   Главное окно закроется автоматически, чтобы программа не оставалась в памяти.

---

## Использование конвертера изображений

1. **Выбор файла**  
   - Нажмите **Выбор файла**.  
   - Выберите изображение.  
   - Имя файла автоматически подтянется в поле `Имя файла`, его можно изменить вручную.  
   - Появится превью изображения.  
   - Отобразятся размер и дата изменения файла.  
   - Формат входного файла будет определен автоматически и заблокирован.

2. **Выбор выходного формата**  
   - Выберите нужный формат в поле `Формат`.  
   - Например, для иконок используйте `ico`.

3. **Конвертация**  
   - Нажмите **Конвертировать**.  
   - Программа выполнит преобразование файла в выбранный формат.  

4. **Сохранение файла**  
   - Нажмите **Сохранить**, выберите папку и имя.  
   - Программа сохранит конвертированный файл в указанное место.  
   - После сохранения обновятся поля с размером и датой изменения.

---

## Использование конвертера аудио

1. **Выбор файла**  
   - Нажмите **Выбор файла**.  
   - Выберите аудиофайл.  
   - Появятся имя, длительность и битрейт.  
   - Формат входного файла определяется автоматически и заблокирован.

2. **Выбор выходного формата**  
   - Выберите желаемый формат для конвертации (`mp3`, `wav`, `flac`, `aac`, `wma`).

3. **Конвертация**  
   - Нажмите **Конвертировать**.  
   - Прогресс будет отображаться в полосе прогресса.  
   - Конвертация может занять некоторое время, в зависимости от размера файла.  

4. **Сохранение файла**  
   - Нажмите **Сохранить**, выберите место и имя для нового файла.  

---

## Дополнительно

- **Перейти к другому модулю**:  
  В меню конвертера можно выбрать переход на другой конвертер (аудио ↔ изображения).  
- **Открыть файл**:  
  После конвертации можно открыть папку с файлом через стандартный проводник.  
- **Язык интерфейса**:  
  Программа автоматически подстраивается под язык системы: русский или английский.  

---

## Советы пользователю

- Для больших аудиофайлов конвертация может занимать несколько секунд. Не закрывайте программу в процессе.  
- При конвертации изображений в ICO лучше использовать квадратные изображения, чтобы получить качественные иконки.  
- Всегда выбирайте место сохранения после конвертации, чтобы не потерять файлы.
